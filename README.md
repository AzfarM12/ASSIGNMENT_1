# Git Lab Assignment

## What I Learned
In this assignment, I learned how to use Git and GitHub to manage project versions. I learned how to initialize a repository, create branches, merge changes, and resolve merge conflicts.

## Challenges Faced
One of the challenges I faced was resolving merge conflicts, as it required careful editing of the file. I also had difficulty understanding how branching works at the beginning.

## How Git Helps in Real Projects
Git helps developers collaborate efficiently by allowing multiple team members to work on different features without affecting the main project. It also helps track changes and improves project organization.
